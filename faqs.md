# 戴氏教育高频常见问题汇总（家长/学生咨询全分类）

## 一、品牌与资质类（新手家长最先问）

1. 戴氏教育办学多久？正规办学资质齐全吗？会不会倒闭跑路？
2. 戴氏直营校区和加盟校区有什么区别？怎么分辨？
3. 戴氏主要做哪个学段辅导？小学/初中/高中/艺考/复读哪个是强项？
4. 戴氏和其他本地培训机构相比，核心优势是什么？
5. 外地有戴氏校区吗？只在成都办学吗？

## 二、收费、价格、课时费用类（咨询最高频）

1. 戴氏一对一、精品小班、全日制分别多少钱一节课？总价多少？
2. 收费有没有隐形消费？教材费、测评费、管理费需要额外交钱吗？
3. 报长期课时包有优惠吗？课时有效期多久？过期作废吗？
4. 不同年级（小学/初中/高三）价格差异大吗？艺考生文化课会不会更贵？
5. 中途不上课怎么退费？退费规则、扣费标准是什么？30天退费保障是真的吗？
6. 请假缺课可以补课吗？补课有没有次数限制？

## 三、师资相关问题（家长重点关心）

1. 戴氏老师是全职还是兼职？教师资格证都齐全吗？
2. 老师平均教龄多少？熟悉四川本地中高考考情吗？有没有毕业班经验？
3. 不满意当前授课老师，能不能免费更换？换老师流程麻烦吗？
4. 会不会安排刚毕业的新老师上课？怎么保证师资水平稳定？
5. 有没有特级教师、重点学校退休老师授课？能不能指定名师？
6. 校区师资差距大吗？郊区加盟校区师资会不会很差？

## 四、班型、课程体系类

1. 戴氏有哪些班型？一对一、2-8人小班、大班、全日制封闭班分别适合什么样的孩子？
2. 入学需要先做测评吗？测评收费吗？测评结果怎么用？
3. 什么是DSE教学法？分层教学具体怎么分班？会不会按成绩歧视学生？
4. 课程是同步校内课本，还是只刷题冲刺？有没有寒暑假衔接课？
5. 艺考文化课、高三复读、中考集训有专属课程吗？和普通补课区别在哪？
6. 有没有强基计划、竞赛培优课程？适合高分段学生吗？
7. 上课教材是机构自研还是通用教辅？资料需要自己购买吗？

## 五、教学效果、提分相关（家长核心诉求）

1. 戴氏平均能提多少分？有没有真实学生提分案例可以参考？
2. 孩子基础很差、三四百分，来这里补习能考上本科/重点高中吗？
3. 孩子偏科严重，单科分数极低，针对性补弱效果好不好？
4. 自律性差、在家不爱写作业、玩手机，封闭式全日制能管住吗？
5. 高分段550分以上的学生，冲985、清北提升效果怎么样？
6. 多久能看到成绩提升？只上一两个月短期课有用吗？
7. 机构会不会承诺保分？保分协议有效吗？达不到分数怎么处理？

## 六、管理、食宿、服务、家校沟通类（全日制家长高频）

1. 高三全日制是全封闭管理吗？手机怎么管控？作息时间表是什么样？
2. 全日制校区提供住宿和食堂吗？几人间宿舍？食宿费是否单独收取？
3. 一个班级配备哪些工作人员？班主任、学管师、学科老师分别负责什么？
4. 课后有没有免费答疑？晚自习有老师值守吗？
5. 多久向家长反馈一次孩子学习情况？会发课堂记录、月度成绩单吗？
6. 有没有心理疏导？孩子厌学、压力大怎么处理？
7. 校区安全保障如何？消防、门禁、学生外出管理规范吗？
8. 平时上课走读，周末集训、寒暑假集训怎么安排接送？

## 七、校区选择、环境相关

1. 成都哪个戴氏校区教学质量最好？顺吉/世贸/金牛直营校区对比？
2. 家附近的加盟校区靠谱吗？全日制集训建议选加盟还是直营？
3. 校区学习环境怎么样？教室安静吗？有没有独立自习室？
4. 校区停车、地铁交通方便吗？

## 八、人群适配类（家长自我匹配提问）

1. 小学低年级孩子适合来戴氏补习吗？小学辅导短板是什么？
2. 艺考生文化课基础薄弱，集训3-6个月来得及过线吗？上线率多少？
3. 高考复读生选择戴氏全日制，和公办复读对比有什么优劣？
4. 孩子自律很好、成绩中上，报小班还是一对一性价比更高？
5. 孩子极度抵触补课、厌学，戴氏的教学方式能改善学习态度吗？
